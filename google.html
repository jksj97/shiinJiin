Ok. 추가 해결점을 모색해서 더 나은 서비스를 시작할게요.

캘린더의 문제점을 분석할게요.

현재 구글 캘린더

  1. 타 웹사이트들과 비교해 문제 분석을 완료했어요.

  2. 현재 다른 웹사이트보다 최소 폰트 1px 이하 본문 사이즈 2px 이하로 작아요.

    1. Naver

      min font 12px

      normal font 14px

    2. Behance

      min font 12px

      normal font 13~14px

    3. YouTube

      min font 12px

      normal font 14px

    4. GitHub

      min font 12px

      normal font 14px

  4. 컨텐츠 영역과 마진을 고려하여 최소 폰트 12px 본문 사이즈 16px로 변환, Button 사이즈 30*30px로 업데이트를 진행할게요.

캘린더의 레이아웃을 변경합니다.

텍스트를 키운 대신 아이템 박스의 height와 마진값을 높여서 사용자의 내용 인식을 높입니다.

사이드 메뉴의 클릭 영역을 40*40px 이지만 디자인 내부의 백그라운드 크기를 제외한
크기는 20px 이하로 인식되기 때문에 터치 영역을 제외한 디자인의 크기를 10px 더 크기를 키웠습니다.

서비스 문제점을 분석할게요.

평일/ 쉬는 날 구분, 카테고리 흐름 분석, 스케쥴 기능 분리 등 문제 분석을 시작합니다.

사용자가 캘린더를 이용하는 목적은 무엇일까요?

  1. 내 스케줄을 정리하기 위해서

  2. 스케줄를 공유해서 팀원들과 일정 조정을 하기 위해서

  3. 앞으로의 일정을 미리 정하기 위해서

  4. 오늘 일정을 계획하기 위해서

  5. 지난 일정에 무엇을 했는지 확인하기 위해서

현재 구글 캘린더의 서비스에는 어떤 기능이 제공되어지나요?

  1. 일정추적 및 메모

    1. 약속 시간대 작성
    2. 휴가, 비공개
      시간예약

  2. 메일함 일정 작성

    1. Gmail에 일일
      일정목록 다운로드
    2. Gmail 받은편지함에
      캘린더 응답 필터링

  3. 팀원과의 공동작업

    1. 일정응답 관리
    2. 내 일정에 다른 사용자 관리/수정 허용
    3. 일정에 그룹 초대

  4. 캘린더의 일정공유

    1. 팀 캘린더 제작, 공유
    2. 동료 캘린더 확인
    3. 회의시간 서치

  5. 제약없는 회의 개최

    1. 회의 일정 예약, 개최
    2. 자료 공유

현재 구글 캘린더에 없는 기능은 어떤 것이 있나요?

스케줄 전체 확인 목록표

일,주,월,년별 확인/키워드 서치로 스케줄 전체를 확인할 수 있어요.

일,주,월,년별로 전체 스케줄 확인

키워드 검색 스케줄 확인

일,월 등 확인/검색 스케줄 확인 = 디테일 정보 확인 불가, 한정된 확인

전체 일정의 디테일을 간략하게 확인할 수 있어야합니다.

  1. 스케쥴의 별도 캘린더 만들기 기능 제공

    현재 구글 캘린더의 ‘내 캘린더’ 기능은 표시 카테고리 추가 기능입니다.
    별도의 다른 캘린더를 만드는 기능이 아니므로 내가 원하는 새로운 캘린더의 목록 구성은 불가능합니다.

  2. 내 일정 목록 전체 보기

    키워드 검색과 날자 주기의 간단한 목록 확인이 가능하지만 전체 일정의 디테일은 확인할 수 없습니다.
    추가로 장기와 단기 일정, 스케쥴의 진행완료/진행도를 표시해 캘린더의 사용성을 높여야 합니다.

  3. 평일/주말 구분

    모든 날자가 단색으로 처리되어 있습니다.(공휴일은 체크박스로 표시) 구분 색이 필요합니다.

  4. 스케쥴 서비스의 세분화

    구글 캘린더의 서비스 사용할 때의 플로우는 심플합니다.
    하지만 그로 인해 서비스의 정확한 서치와 이용, 이해가 떨어집니다. 정확한 구분이 필요합니다.

  5. 알림 기능 시각화

    정확한 일정 기능 수행을 위해 알람 카테고리를 ‘사용안함/소리알림/팝업알림(추가)’으로 구성합니다.

업데이트 내역의 분석이 끝났습니다.
진행 목록 확인 후 업데이트를 진행합니다.

업데이트 내역을 확인해 주십시오.

  1. 사이즈 변경으로 사용성 확대

    1. 폰트 사이즈 변경: 최소 사이즈 12px/평균 사이즈 14px
    2. 캘린더 날자 버튼 영역 확대: 30*24px > 30*30px

  2. 평일/공휴일 구분색 표시 > 명료한 스케쥴 선정 가능

    1. 평일 #151515/주말,공휴일 #E74343 (지난 일자 평일(#A5A8AC)/주말,공휴일(#FFCECE))

  3. 스케쥴 서비스 목록 별도 메뉴바 Icon 구성 > 서비스 제공 직관성 우선/텍스트 과다 디자인 중화

    1. 월,일일 스케쥴 페이지 별도 구성/스케쥴 목록표 구성 (그 외 메모,공유 등 기본 구성 유지)
    2. 스케쥴 확인에 방해되지 않도록 글라스피즘 적용 > 공감각 효과
    3. 아이콘 특성에 맞는 모션적용으로 경험성 증가

  4. 알람기능 추가 > 사용안함/소리/모달창(시각효과) > 시각 확인 가능한 알람창으로 알람 기능 강화

    1. Google Style의 일러스트 디자인 알람창 구성
